# Homecare Robot

## Summary

Development of new cleaning robot as part of the BOSCH consumer product family

## Description

The project is a joint effort of BSH Homecare Robotics, BOSCH Corporate Research, BOSCH Global Software Technologies to develop a portfolio of BOSCH cleaning robots with cutting-edge technologies ready for mass production in manufactory.

## Contributions

- Software Development of robotic navigation & behavior software
- Funding of connectivity with smart IoT ecosystem Home Connect [^1]
- Requirement management
