# Aerial RFID Scanning for Warehouses

## Summary

Drone solution of efficient inventory management in warehouses.

## Description

Drones traverse the warehouse shelves based on known route graphs and scan RFID tags for inventory. The drone can fly around obstacles and re-route on the graph level.

## Contributions

- Funding DevOps infrastructure
- Software integration
- ROS2 migration
